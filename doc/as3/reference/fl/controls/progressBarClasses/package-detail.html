fl.controls.progressBarClassesClasses
 

The fl.controls.progressBarClasses package contains classes that are specific to the ProgressBar component. Currently, this package contains only the default IndeterminateBar class, which controls the display of the ProgressBar when it is indeterminate.


Classes
 ClassDescription
 IndeterminateBar The IndeterminateBar class handles the drawing of the progress bar component when the size of the source that is being loaded is unknown.
